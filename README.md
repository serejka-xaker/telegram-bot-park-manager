# Проект: Микросервисная SaaS-платформа для управления парком Telegram-ботов

**Декабрь 2025 | Фриланс**

- Спроектировал архитектуру Telegram-бота на базе aiogram 3, состоящую из управляющего мастер-бота и динамически создаваемых дочерних-ботов, запускаемых в отдельных процессах с передачей изолированной конфигурации.
- Создал админ-панель мастер-бота с пошаговым управлением (FSM), Inline-клавиатурами, контролем жизненного цикла дочерних ботов, модерацией заявок и разграничением ролей.
- Спроектировал архитектуру с уникальными токенами и API-ключами для каждого дочернего бота, изоляцией данных и независимыми клиентскими транзакциями.
- Создал асинхронную модель БД на PostgreSQL с хранением конфигураций ботов, данных мерчантов и транзакций, реализовав асинхронные CRUD-операции.
- Интегрировал REST API PayAssist для работы с инвойсами, статусами ордеров, балансом и обработкой ошибок API в функционал дочерних ботов. 
- Реализовал асинхронное логирование ошибок дочерних ботов в БД с изоляцией от основного цикла и защитой сервиса от падений.
- Обеспечил отказоустойчивый деплой через Docker Compose с контейнерами приложения и PostgreSQL, проверкой состояния БД, изолированной сетью и оптимизированными образами.

### Стек
`Python 3.11`, `aiogram 3.x`, `Asyncio`, `SQLAlchemy 2.0 Async`, `PostgreSQL 16`, `Docker`, `Docker Compose`, `REST API`, `RBAC`, `FSM`, `Multi-tenancy`, `Subprocess`
